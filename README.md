# Drugtionary

## Start Backend Server
```
$ npm install express
$ cd backend
$ npm run serve
```

## Start Frontend Server
```
$ npm install @vue/cli
$ npm install bulma axios vue-router
$ cd frontend
$ npm run serve
```
